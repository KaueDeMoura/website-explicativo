NodeJs - Básico

O papel do Node.js no Backend

O Node.js é uma plataforma construída sobre o motor de execução JavaScript do Chrome, o V8. Ele permite criar servidores eficientes e escaláveis, graças ao seu modelo de execução assíncrono e orientado a eventos. Isso significa que o Node.js é excelente para aplicações em tempo real e altamente escaláveis.

Utilização de Módulos Principais

Utilização de Módulos Principais

Node.js possui diversos módulos nativos que ajudam a criar funcionalidades de servidor. Abaixo estão alguns dos principais módulos:

Módulo HTTP

Este módulo permite criar e manipular servidores HTTP:

const http = require('http');
const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello World');
});
server.listen(3000, () => console.log('Servidor rodando na porta 3000'));

Módulo FS

Este módulo permite manipular o sistema de arquivos (File System):

const fs = require('fs');
fs.readFile('arquivo.txt', 'utf8', (err, data) => {
  if (err) throw err;
  console.log(data);
});

Módulo PATH

Este módulo ajuda a manipular e resolver caminhos de arquivos:

const path = require('path');
const caminhoCompleto = path.join(__dirname, 'arquivos', 'imagem.png');
console.log(caminhoCompleto);

Estrutura de um Aplicativo Express.js

Estrutura de um Aplicativo Express.js

Express.js é um framework web minimalista que facilita o desenvolvimento com Node.js, proporcionando uma estrutura flexível para criação de aplicações e APIs.

Configurando um Servidor Express

const express = require('express');
const app = express();
const PORT = 3000;

app.get('/', (req, res) => {
  res.send('Bem-vindo ao servidor Express!');
});

app.listen(PORT, () => console.log(`Servidor rodando na porta ${PORT}`));

Criando Rotas

Em um servidor Express, você pode definir várias rotas para diferentes requisições HTTP:

app.get('/usuarios', (req, res) => {
  res.send('Listando todos os usuários');
});

app.post('/usuarios', (req, res) => {
  res.send('Criando um novo usuário');
});

Integração com Banco de Dados

Integração com Banco de Dados

Node.js permite integração com diversos bancos de dados usando bibliotecas. Aqui estão dois exemplos populares:

Integração com MongoDB usando Mongoose

const mongoose = require('mongoose');
mongoose.connect('mongodb://localhost:27017/meuBanco', { useNewUrlParser: true, useUnifiedTopology: true });

const Usuario = mongoose.model('Usuario', { nome: String, idade: Number });
const novoUsuario = new Usuario({ nome: 'João', idade: 25 });

novoUsuario.save().then(() => console.log('Usuário salvo!'));

Integração com PostgreSQL usando Sequelize

const { Sequelize, DataTypes } = require('sequelize');
const sequelize = new Sequelize('postgres://usuario:senha@localhost:5432/meuBanco');

const Usuario = sequelize.define('Usuario', {
  nome: DataTypes.STRING,
  idade: DataTypes.INTEGER
});

sequelize.sync().then(() => console.log('Tabela criada!'));

Práticas de Segurança

Práticas de Segurança

A segurança é fundamental no desenvolvimento de back-end. Utilizar middlewares para autenticação e autorização ajuda a proteger suas rotas.

Middleware de Autenticação

Um exemplo básico de middleware para verificar autenticação:

function autenticar(req, res, next) {
  if (req.isAuthenticated()) {
    return next();
  }
  res.status(401).send('Acesso negado');
}

app.use('/rotaProtegida', autenticar, (req, res) => {
  res.send('Acesso concedido');
});
