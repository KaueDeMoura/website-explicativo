React - Básico

1. Estrutura de Componentes

1. Estrutura de Componentes

React é baseado em componentes, que são pedaços reutilizáveis de código que representam partes da interface de usuário. Cada componente React é uma função JavaScript ou uma classe que retorna um pedaço da UI a ser renderizada. Eles permitem a criação de interfaces dinâmicas, compondo a aplicação em blocos menores e mais fáceis de gerenciar.

                
function MeuComponente() {
    return (
        <div>
            <h1>Olá, Mundo!</h1>
            <p>Este é um componente em React.</p>
        </div>
    );
}

2. Hooks: useState e useEffect

2. Hooks: useState e useEffect

Hooks são funções especiais que permitem o uso de estado e outros recursos do React em componentes funcionais. Os dois hooks mais usados são:

useState

O useState permite adicionar estado a um componente funcional. Ele retorna um valor de estado e uma função para atualizá-lo.

                
import React, { useState } from 'react';

function Contador() {
    const [contador, setContador] = useState(0);

    return (
        <div>
            <p>Contador: {contador}</p>
            <button onClick={() => setContador(contador + 1)}>Incrementar</button>
        </div>
    );
}

useEffect

O useEffect permite executar efeitos colaterais em componentes, como buscar dados de uma API ou configurar um intervalo. Ele é executado após a renderização e pode depender de variáveis específicas.

                
import React, { useEffect } from 'react';

function ExemploEfeito() {
    useEffect(() => {
        console.log('O componente foi montado');

        return () => {
            console.log('O componente foi desmontado');
        };
    }, []);

    return <div>Veja o console para o efeito</div>;
}

3. JSX

3. JSX

JSX é uma extensão de sintaxe para JavaScript que permite escrever código similar ao HTML dentro do JavaScript. Ele facilita a criação de interfaces em React, permitindo misturar lógica JavaScript com elementos HTML de forma clara e concisa.

                
import React from 'react';

function Saudacao() {
    const nome = 'João';
    return <h1>Olá, {nome}!</h1>;
}

4. Navegação com react-router-dom

4. Navegação com react-router-dom

O React Router é uma biblioteca que permite navegação entre páginas em uma aplicação React. Com ele, podemos definir rotas que renderizam diferentes componentes.

                
import React from 'react';
import { BrowserRouter as Router, Route, Link, Switch } from 'react-router-dom';

function App() {
    return (
        <Router>
            <nav>
                <Link to="/">Home</Link>
                <Link to="/sobre">Sobre</Link>
            </nav>

            <Switch>
                <Route exact path="/" component={Home} />
                <Route path="/sobre" component={Sobre} />
            </Switch>
        </Router>
    );
}

function Home() {
    return <h1>Página Inicial</h1>;
}

function Sobre() {
    return <h1>Sobre Nós</h1>;
}

5. Gerenciamento de Estado Global: Redux e Context API

5. Gerenciamento de Estado Global: Redux e Context API

Para gerenciar o estado em toda a aplicação, o React permite a integração com bibliotecas como Redux ou Context API. Isso é útil em aplicativos maiores onde muitos componentes precisam acessar e atualizar o mesmo estado.

Context API

O Context API é uma ferramenta interna do React para compartilhar dados entre componentes sem a necessidade de passar props manualmente em cada nível.

                
import React, { createContext, useContext, useState } from 'react';

const TemaContexto = createContext();

function App() {
    const [tema, setTema] = useState('claro');

    return (
        <TemaContexto.Provider value={{ tema, setTema }}>
            <MinhaPagina />
        </TemaContexto.Provider>
    );
}

function MinhaPagina() {
    const { tema, setTema } = useContext(TemaContexto);

    return (
        <div style={{ backgroundColor: tema === 'claro' ? '#fff' : '#333' }}>
            <p>Tema atual: {tema}</p>
            <button onClick={() => setTema(tema === 'claro' ? 'escuro' : 'claro')}>Mudar Tema</button>
        </div>
    );
}
