HTML - Básico

Estruturais

As tags estruturais são usadas para definir a estrutura básica de um documento HTML.

1.

Define o início e o fim de um documento HTML.

        
        html
            head
                meta charset="UTF-8"
                titleMeu Título/title
            /head
            body
                pConteúdo da página./p
            /body
        /html

2. head

Contém metadados, links para estilos, scripts e o título da página.

        
        head
            meta charset="UTF-8"
            titleTítulo da Página/title
            link rel="stylesheet" href="styles.css"
            script src="script.js"/script
        /head

3. body

Contém o conteúdo visível da página, como texto, imagens e vídeos.

        
        body
            h1Título Principal/h1
            pEste é um parágrafo de exemplo./p
        /body

4. div

Usado para agrupar blocos de conteúdo. Geralmente usado para estilização com CSS.

        
        div class="container"
            pEste é um bloco de conteúdo dentro de um div./p
        /div

5. span

Usado para estilizar partes específicas de um texto sem quebrar a linha. Similar ao div, mas em nível inline.

        
        pEste é um texto normal, mas span style="color: red;"este texto está em vermelho/span./p

Textuais

As tags textuais são usadas para formatar e estruturar o texto dentro de uma página HTML.

1. h1 - h6

Tags de cabeçalho que variam de h1 (mais importante) a h6 (menos importante).

        
        h1Cabeçalho de Nível 1/h1
        h2Cabeçalho de Nível 2/h2
        h3Cabeçalho de Nível 3/h3
        h4Cabeçalho de Nível 4/h4
        h5Cabeçalho de Nível 5/h5
        h6Cabeçalho de Nível 6/h6

2. p

Usado para definir um parágrafo de texto.

        
        pEste é um parágrafo simples que contém algum texto para demonstração./p

3. a

Define um hiperlink, que pode ser usado para navegar para outra página ou parte da mesma página.

        
        a href="https://exemplo.com" target="_blank"Clique aqui para visitar o exemplo.com/a

4. strong

Usado para enfatizar texto, geralmente exibido em negrito.

        
        pAqui está um texto normal, mas strongeste texto está em negrito/strong./p

5. em

Usado para enfatizar texto, geralmente exibido em itálico.

        
        pAqui está um texto normal, mas emeste texto está em itálico/em./p

Mídia

As tags de mídia são usadas para incorporar imagens, áudio e vídeo em uma página HTML.

1. img

Usado para exibir uma imagem. O atributo src define o caminho da imagem.

        
        img src="imagem.jpg" alt="Descrição da imagem" width="200" height="150"

2. Audio

Usado para incorporar áudio. Pode incluir controles para reprodução.

        
        audio controls
            source src="audio.mp3" type="audio/mp3"
        /audio

3. Video

Usado para incorporar vídeo. Pode incluir controles para reprodução.

        
        video controls width="300"
            source src="video.mp4" type="video/mp4"
        /video

Formulários

1. form

Define um formulário HTML que contém elementos de entrada (input) e botões.

            label for="nome"Nome:/label
            input type="text" id="nome" name="nome" required
            br
            label for="email"E-mail:/label
            input type="email" id="email" name="email" required
            br
            button type="submit"Enviar/button
        /form

Tabelas

As tags de tabelas são usadas para exibir dados em formato de grade ou tabela.

1. table

Define uma tabela HTML que contém linhas (tr) e células (td ou th).

        
        table border="1"
            tr
                thNome/th
                thIdade/th
                thCidade/th
            /tr
            tr
                tdJoão/td
                td25/td
                tdSão Paulo/td
            /tr
            tr
                tdMaria/td
                td30/td
                tdRio de Janeiro/td
            /tr
        /table

Exemplo de como fica na página:

Nome Idade Cidade
João 25 São Paulo
Maria 30 Rio de Janeiro

2. tr

Define uma linha em uma tabela.

                    
        
        tr
            tdDados 1/td
            tdDados 2/td
        /tr

Exemplo de como fica na página:

Dados 1 Dados 2

3. td

Define uma célula de dados em uma tabela.

        
        tdDados da Célula/td

Exemplo de como fica na página:

Dados da Célula

4. th

Define uma célula de cabeçalho em uma tabela, que geralmente aparece em negrito e é centralizada.

        
        thCabeçalho/th

Exemplo de como fica na página:

Cabeçalho

Semânticas

As tags semânticas são usadas para dar significado ao conteúdo de uma página HTML, melhorando a acessibilidade e SEO.

1. header

Usado para definir o cabeçalho de uma página ou seção, geralmente contém o logotipo, menu de navegação ou título da página.

            
            header
                h1Bem-vindo ao Meu Site/h1
                nav
                    a href="#home"Início/a
                    a href="#sobre"Sobre/a
                    a href="#contato"Contato/a
                /nav
            /header

Exemplo de como fica na página:

Bem-vindo ao Meu Site

Início Sobre Contato

2. footer

Usado para definir o rodapé de uma página ou seção, geralmente contém informações de direitos autorais, links de política de privacidade ou contatos.

            
            footer
                p© Site dos Guri./p
                p
                    a href="https://facebook.com"Facebook/a |
                    a href="https://twitter.com"Twitter/a
                /p
            /footer

Exemplo de como fica na página:

© Site dos Guri.

Facebook | Twitter

3. article

Usado para definir um conteúdo independente e de relevância, como artigos de notícias, postagens de blog ou comentários.

            
            article
                h2Título do Artigo/h2
                pEste é o conteúdo do artigo. Pode incluir texto, imagens, vídeos, etc./p
            /article

Exemplo de como fica na página:

Título do Artigo

Este é o conteúdo do artigo. Pode incluir texto, imagens, vídeos, etc.

4. section

Usado para definir uma seção dentro de um documento. Pode conter cabeçalhos, artigos, e outras divisões de conteúdo.

            
            section
                h2Seção Principal/h2
                pEsta é uma seção do documento. Pode conter várias tags e conteúdo diverso./p
            /section

Exemplo de como fica na página:

Seção Principal

Esta é uma seção do documento. Pode conter várias tags e conteúdo diverso.

5. nav

Usado para definir uma seção de navegação que contém links para outras partes da página ou site.

            
            nav
                ul
                    li a href="#home"Início/a /li
                    li a href="#servicos"Serviços/a /li
                    li a href="#contato"Contato/a /li
                /ul
            /nav

Exemplo de como fica na página:

  • Início
  • Serviços
  • Contato
