JS - Básico

Principais Utilizações do JavaScript no Frontend

JavaScript é uma linguagem de programação poderosa usada para adicionar interatividade às páginas web. Abaixo estão algumas das principais funcionalidades:

Manipulação de DOM

Manipulação de DOM

Exemplos de funções utilizadas para manipular o DOM:

  • document.getElementById: Seleciona um elemento pelo seu ID.
  • document.querySelector: Seleciona o primeiro elemento que corresponde a um seletor CSS.
  • document.createElement: Cria um novo elemento HTML.
  • element.appendChild: Adiciona um elemento como filho de outro elemento.

Eventos

Eventos

JavaScript permite capturar e responder a eventos do usuário, como:

  • addEventListener: Adiciona um ouvinte de eventos a um elemento.
  • Eventos comuns: click, submit, mouseover, keydown.

Manipulação de Dados

Manipulação de Dados

JavaScript possui várias ferramentas para manipulação de dados, como:

  • Arrays: Métodos como map, filter, reduce.
  • Objetos: Criação e acesso a propriedades de objetos.
  • Funções: Declaração, expressão e arrow functions.

Fetch API

Fetch API

JavaScript permite realizar requisições HTTP assíncronas usando:

  • fetch: API para fazer requisições HTTP.
  • Promises e Async/Await: Para trabalhar com operações assíncronas de forma mais legível.
