CSS - Básico

Propriedades de CSS

As propriedades de CSS são usadas para controlar o layout e a aparência dos elementos em uma página web.

1. color

Define a cor do texto de um elemento.

                
                p {
                    color: blue;
                }

Este texto é azul.

2. background-color

Define a cor de fundo de um elemento.

                
                div {
                    background-color: yellow;
                }
Este div tem fundo amarelo.

3. font-size

Controla o tamanho da fonte de um texto.

                
                h1 {
                    font-size: 24px;
                }

Este é um título com tamanho de fonte 24px.

4. margin

Define o espaço externo ao redor de um elemento.

                
                div {
                    margin: 20px;
                }
Este div tem uma margem de 20px ao redor.

5. padding

Define o espaço interno entre o conteúdo de um elemento e suas bordas.

                
                div {
                    padding: 15px;
                }
Este div tem um padding de 15px.

6. border

Define as bordas ao redor de um elemento.

                
                p {
                    border: 2px solid red;
                }

Este parágrafo tem uma borda vermelha.

7. display

Controla o comportamento de exibição de um elemento. Pode ser block, inline, flex, entre outros.

                
                div {
                    display: flex;
                }

8. position

Define o método de posicionamento de um elemento. Pode ser static, relative, absolute, fixed ou sticky.

                
                div {
                    position: absolute;
                    top: 50px;
                    left: 100px;
                }
Este div está posicionado absolutamente.

9. text-align

Controla o alinhamento do texto dentro de um elemento. Pode ser left, right, center, ou justify.

                
                p {
                    text-align: center;
                }

Este texto está centralizado.

10. width e height

Definem a largura e a altura de um elemento.

                
                img {
                    width: 200px;
                    height: auto;
                }

11. flex

Usado em conjunto com display: flex, define o crescimento, encolhimento e base do tamanho dos itens flexíveis.

                
                div {
                    display: flex;
                }
                div.item {
                    flex: 1;
                }

12. grid

Define um layout de grade para posicionar elementos em linhas e colunas.

                
                div {
                    display: grid;
                    grid-template-columns: 1fr 2fr;
                }
