Guia CSS - Propriedades e Seletores

Seletores CSS

Seletores CSS

CSS permite estilizar elementos específicos na página com os seletores abaixo:

  • .classe: Seleciona todos os elementos com uma determinada classe. .exemplo
  • #id: Seleciona um elemento único com um ID específico. #exemplo
  • Elemento: Seleciona todos os elementos HTML de um tipo específico, como p ou h1
  • [atributo]: Seleciona elementos com um atributo específico. [type="text"]
  • :hover: Pseudo-classe que aplica o estilo quando o mouse está sobre o elemento.
  • ::before: Pseudo-elemento que insere conteúdo antes do conteúdo de um elemento.

Propriedades de Texto

Propriedades de Texto

  • color: Define a cor do texto. color: red;
  • font-size: Ajusta o tamanho da fonte. font-size: 16px;
  • font-weight: Controla a espessura do texto (negrito). font-weight: bold;
  • line-height: Define o espaçamento entre linhas de texto. line-height: 1.5;
  • text-align: Alinha o texto no elemento. text-align: center;

Propriedades de Layout

Propriedades de Layout

  • display: Controla o tipo de exibição de um elemento (block, inline, flex, grid). display: flex;
  • position: Define a posição de um elemento (relative, absolute, fixed, sticky). position: absolute;
  • top, right, bottom, left: Controla a posição de um elemento posicionado. top: 20px;
  • float: Posiciona elementos à esquerda ou direita. float: left;
  • clear: Controla o comportamento de elementos flutuantes. clear: both;

Propriedades do Modelo de Caixa (Box Model)

Propriedades do Modelo de Caixa (Box Model)

  • margin: Define o espaçamento externo ao redor do elemento. margin: 10px;
  • padding: Define o espaçamento interno dentro do elemento. padding: 15px;
  • border: Adiciona uma borda ao redor do elemento. border: 1px solid black;
  • width: Define a largura do elemento. width: 100px;
  • height: Define a altura do elemento. height: 200px;

Propriedades de Fundo (Background)

Propriedades de Fundo (Background)

  • background-color: Define a cor de fundo. background-color: blue;
  • background-image: Define uma imagem de fundo. background-image: url('imagem.jpg');
  • background-size: Ajusta o tamanho da imagem de fundo. background-size: cover;
  • background-repeat: Controla a repetição da imagem de fundo. background-repeat: no-repeat;

Propriedades de Flexbox e Grid

Propriedades de Flexbox e Grid

  • display: flex: Habilita o uso de Flexbox. display: flex;
  • justify-content: Alinha itens horizontalmente em um contêiner flex. justify-content: center;
  • align-items: Alinha itens verticalmente em um contêiner flex. align-items: center;
  • display: grid: Habilita o uso de Grid. display: grid;
  • grid-template-columns: Define as colunas em um contêiner grid. grid-template-columns: 1fr 1fr;
  • grid-gap: Define o espaçamento entre itens de um contêiner grid. grid-gap: 10px;
