HTML - Básico

Estruturais

1. Estruturais

As tags estruturais são usadas para definir a estrutura básica de um documento HTML.

Define o início e o fim de um documento HTML.

        
        html
            head
                meta charset="UTF-8"
                titleMeu Título/title
            /head
            body
                pConteúdo da página./p
            /body
        /html

2. head

Contém metadados, links para estilos, scripts e o título da página. 

        
        head
            meta charset="UTF-8"
            titleTítulo da Página/title
            link rel="stylesheet" href="styles.css"
            script src="script.js"/script
        /head

3. body

Contém o conteúdo visível da página, como texto, imagens e vídeos. 

        
        body
            h1Título Principal/h1
            pEste é um parágrafo de exemplo./p
        /body

4. div

Usado para agrupar blocos de conteúdo. Geralmente usado para estilização com CSS. 

        
        div class="container"
            pEste é um bloco de conteúdo dentro de um div./p
        /div

5. span

Usado para estilizar partes específicas de um texto sem quebrar a linha. Similar ao div, mas em nível inline. 

        
        pEste é um texto normal, mas span style="color: red;"este texto está em vermelho/span./p

Textuais

Textuais

As tags textuais são usadas para formatar e estruturar o texto dentro de uma página HTML.

1. h1 - h6

Tags de cabeçalho que variam de h1 (mais importante) a h6 (menos importante). 

              
              h1Cabeçalho de Nível 1/h1
              h2Cabeçalho de Nível 2/h2
              h3Cabeçalho de Nível 3/h3
              h4Cabeçalho de Nível 4/h4
              h5Cabeçalho de Nível 5/h5
              h6Cabeçalho de Nível 6/h6

2. p

Usado para definir um parágrafo de texto.

              
              pEste é um parágrafo simples que contém algum texto para demonstração./p

3. a

Define um hiperlink, que pode ser usado para navegar para outra página ou parte da mesma página. 

              
              a href="https://exemplo.com" target="_blank"Clique aqui para visitar o exemplo.com/a

4. strong

Usado para enfatizar texto, geralmente exibido em negrito.

              
              pAqui está um texto normal, mas strongeste texto está em negrito/strong./p

5. em

Usado para enfatizar texto, geralmente exibido em itálico.

              
              pAqui está um texto normal, mas emeste texto está em itálico/em./p

Mídia

Mídia

As tags de mídia são usadas para incorporar imagens, áudio e vídeo em uma página HTML.

1. img

Usado para exibir uma imagem. O atributo src define o caminho da imagem. 

                  
                  img src="imagem.jpg" alt="Descrição da imagem" width="200" height="150"

2. Audio

Usado para incorporar áudio. Pode incluir controles para reprodução. 

                  
                  audio controls
                      source src="audio.mp3" type="audio/mp3"
                  /audio

3. Video

Usado para incorporar vídeo. Pode incluir controles para reprodução. 

                  
                  video controls width="300"
                      source src="video.mp4" type="video/mp4"
                  /video
